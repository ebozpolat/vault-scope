VaultScope API Documentation

SaaS + On-Prem Crypto Treasury APIs

Authentication

Authenticate using an API key passed as a bearer token in the Authorization header.

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /wallets/balances

Returns real-time balances from all tracked wallets and vaults.

GET /nav

Returns current Net Asset Value (NAV) with asset-level breakdown.

POST /alerts

Create or update alert rules for asset thresholds, vault events, or concentration risk.

GET /audit/logs

Download or filter audit logs for compliance reporting.

POST /rebalance-rules

Define automated rules to rebalance assets across wallets or custodians.

Webhooks

  • balance.alert.triggered — triggered when thresholds are crossed
  • wallet.rebalanced — after an automated fund movement
  • user.access.changed — logs permission or TAP rule changes
